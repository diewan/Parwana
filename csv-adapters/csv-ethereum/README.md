# csv-ethereum

Ethereum adapter for the CSV Protocol — implements `SealProtocol` and `ChainBackend` traits for Ethereum seal protocols.

## Features

- **rpc** — Ethereum JSON-RPC support
- **real-groth16** — Real Groth16 proof verification
- **quorum** — Multi-node quorum verification
- **production** — Production configuration
