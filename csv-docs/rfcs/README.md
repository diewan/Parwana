# CSV Protocol RFCs

This directory contains Request for Comments (RFCs) for the CSV Protocol.

## RFC Process

All protocol changes MUST go through the RFC process. No protocol change without RFC.

## RFC List

- RFC-0001: Canonical Serialization
- RFC-0002: Proof Taxonomy
- RFC-0003: Finality Model
- RFC-0004: Replay Model
- RFC-0005: Content Model
- RFC-0006: Contract ABI
- RFC-0007: Hash Registry
- RFC-0008: State Machine
- RFC-0009: Schema Registry
- RFC-0010: Adapter Capabilities

## RFC Template

```markdown
# RFC-XXXX: [Title]

## Status
[Proposed | Accepted | Rejected | Deprecated]

## Motivation
Why is this change needed?

## Proposed Change
Detailed description of the change.

## Rationale
Why this approach over alternatives?

## Impact
What breaks? What needs migration?

## Alternatives
What other approaches were considered?

## Unresolved Questions
What remains to be decided?
```
